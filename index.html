How To Install Pi-Apps X86

To Install Pi-Apps X86 Open Terminal And run the following: 

                                
                                    git clone https://github.com/Pi-Apps-Recreates/pi-apps-x86.git 
                                    chmod +x $HOME/pi-apps-x86/install 
                                    bash $HOME/pi-apps-x86/install

About This Project

This Project Was Made For Raspberry Pi Users To Have A Familer Appstore If They Swapped To Linux PC's With amd Or i386 Processers. This Couldn't Of Happend If Botspot Didnt Make Pi-Apps
