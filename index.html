How To Install Pi-Apps X86

To Install Pi-Apps X86 Open Terminal And run the following:

git clone https://github.com/Pi-Apps-Recreates/pi-apps-x86.git chmod +x $HOME/pi-apps-x86/install
bash $HOME/pi-apps-x86/install

About This Project

This Project was made for Raspberry Pi users to have a familar Appstore when they swapped to Linux PC's with amd64 Or i386 processors. This Project wouldn't have been possible without Botspot making Pi-Apps
